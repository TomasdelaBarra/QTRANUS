Data

This part of the program is used to enter all the necessary information for the TRANUS Model. In this manual we are using the model of Swindon as an example. This model and all related data may be downloaded from:

TRANUS site

To begin the process, a simple window is presented with three buttons: Zones, Network, Matrix.

In (1) the scenario tree is displayed as a reference, and in (2) manage the list of created layers.

File

Jumps to the Zones Window, where zone-based maps are produced, such as population by zone, employment density by zone, etc.

Input File

Jumps to the Zones Window, where zone-based maps are produced, such as population by zone, employment density by zone, etc.

Project

Jumps to the Zones Window, where zone-based maps are produced, such as population by zone, employment density by zone, etc.

Land Use

Jumps to the Network Maps Window to produce maps on a link basis, such as vehicle volumes, speeds, level of service, and so on.

Transport

Jumps to the Matrix Flow Maps Window, where there are tools to produce flow maps from O-D matrix data, such as trips to work, or heavy freight.
